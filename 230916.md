# django | React 연동

request : 클라이언트가 서버에게 정보를 (줘라, 받아라, 삭제해라, 갱신해라.. 등등) 요청하는 것

response : request를 받은 서버가 클라이언트에게 제공하는 응답.

Server : 정보를 제공하는 컴퓨터

Client : 정보를 받는 컴퓨터

FrontEnd : 정보를 "보여주는" 파트

BackEnd : 정보를 "제공하는" 파트

Domain : 특정 ip 주소를 인간이 알아보기 쉬운 단어 및 문자열의 형태로 표현되는 이름

ip : 컴퓨터마다 등록되는 고유식별번호. 주민번호 같은 것
